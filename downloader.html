01
Instale as dependências (só na primeira vez)

Abra o Terminal (CMD + Espaço → "Terminal") e rode os comandos abaixo um por um:

pip3 install yt-dlp brew install ffmpeg

Não tem o Homebrew? Instale primeiro em brew.sh

02
Baixe o servidor

Baixe o arquivo server.py e coloque em qualquer pasta (ex: Downloads).

python3 server.py
